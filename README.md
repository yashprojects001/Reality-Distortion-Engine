## 🌌 Concept

**Reality Distortion Engine** is an experimental interactive experience where:

* The UI slowly **mutates over time**
* Text becomes **unstable**
* Elements behave **unpredictably**
* The user begins to question:

  > *“Was it always like this?”*

---

## 🧩 Features

✨ **Progressive Distortion**

* Interface degrades gradually
* No sudden jumps — only subtle corruption

🌀 **Text Mutation Engine**

* Characters randomly distort
* Messages evolve into unsettling statements

🖱️ **Cursor-Based Reality Shift**

* UI reacts to your movement
* Creates a sense of “being watched”

🔘 **Unstable Controls**

* Buttons drift slightly
* Interaction feels unreliable

🎨 **Dynamic Background Drift**

* Colors shift with distortion level
* Visual discomfort increases over time
